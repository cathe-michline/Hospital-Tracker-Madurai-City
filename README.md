**Hospital Tracker for Madurai City** is a full-stack web application that allows users to book hospital appointments online and enables hospitals to manage and view patient bookings efficiently.

---

## ğŸ“ Description

This web application provides a simple and user-friendly platform for patients to:
- Register and log in using their email  
- View a list of hospitals in Madurai City  
- Book appointments by entering patient details such as name, age, gender, hospital, and preferred time  
- View booked appointments on their profile page  

Hospitals are provided with secure login credentials to:
- View all patient appointments  
- Access patient details including name, age, gender, and appointment time  
- Manage daily bookings efficiently
